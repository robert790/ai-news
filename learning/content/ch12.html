A RAG chatbot over your own documents.

Pick any domain — your old company's PDFs, Romanian legal docs, your hobby's manuals. Build a chatbot that answers questions using those documents, not the model's general knowledge. Must include: chunking strategy, embedding choice, retrieval eval, hallucination guardrails.

Stack: LangChain or LlamaIndex · ChromaDB or Qdrant · OpenAI or DeepSeek API · Streamlit for UI

⏱ ~2 weekends
2

An AI agent that does a real multi-step task.

Anything with 3+ steps that uses tools. Examples: a research agent that searches the web, summarizes findings, writes a report. A sales agent that reads a CRM, drafts emails, schedules follow-ups. A travel agent that finds flights, compares hotels, books them.

Stack: smolagents or LangChain agents · tool-calling · function definitions · error handling

⏱ ~1 week
3

A fine-tuned domain model with eval harness.

Take an open model (Llama 3.2, Qwen 2.5, or Mistral). Fine-tune it on 1,000–10,000 examples from a domain you care about. Build an evaluation set that compares your model's output to a baseline. Show the improvement with real numbers.

Stack: Hugging Face TRL · Axolotl or Unsloth · LLaMA-Factory · Weights & Biases for tracking

⏱ ~2 weeks
4

A deployed AI app with real users.

Take any of the above and put it online with a real URL, real auth, real database. Even 10 users counts. The deployment story — Docker, monitoring, error handling, scaling — is what separates engineers from hobbyists.

Stack: FastAPI · Docker · Hugging Face Spaces / Vercel / Railway · Sentry for errors · PostHog for analytics

⏱ ~1 week after project 1–3
5

An AI evaluation suite that you'd defend in an interview.

Build a small framework that tests an AI system's quality on dimensions that matter: factual accuracy, refusal behavior, latency, cost, hallucination rate. Run it against 3 different models or prompts. Present the results as a dashboard or report. This is the rarest skill in 2026.

Stack: Python · your favorite eval library (RAGAS, DeepEval, custom) · a viz tool (Streamlit, Plotly)

⏱ ~1 week
